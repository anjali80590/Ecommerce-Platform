# Project Name
 Ecommerce Platform

## Live Demo
Check out the deployed app here: [Link](https://ecommerce-platform-xpfl.onrender.com/)


## Description
This is a Node.js project that seeds a MongoDB database with sample products using Faker.js and provides a product management backend. 

## Features
- Seed MongoDB with 1000 sample products
- Products have attributes like title, description, price, rating, stock, metadata
- Uses Mongoose for MongoDB integration
- Environment variables support using `.env`

## Tech Stack
- Node.js
- Express.js 
- MongoDB & Mongoose
- Faker.js for generating sample data
- dotenv for environment variables
